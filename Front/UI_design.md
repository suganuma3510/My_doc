# アトミックデザイン
パーツ・コンポーネント単位で定義していく UI デザイン手法。  
製品の各モジュールを作成した後、それらを繋げて完成品を作る手法。  
アトミックデザインにおけるコンポーネントがいくつも組み合わさったような製品のパーツをモジュールという。

メリット
各コンポーネントが簡単に再利用、編集、統合できるため、プロダクトの拡充が容易に行える。
